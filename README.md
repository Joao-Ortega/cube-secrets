# Cube Secrets 🧊

A modern, high-performance web platform dedicated to the speedcubing community. Cube Secrets acts as a curated affiliate storefront for Rubik's cubes and accessories, seamlessly integrated with premium digital courses for players of all levels.

## 🚀 Tech Stack

This project was built with a modern front-end architecture focused on scalability, performance, and clean code:

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Library:** [React](https://react.dev/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling & UI:** [Material UI (MUI v6)](https://mui.com/)

## ✨ Key Features

* **Advanced Data Filtering:** Dynamic cross-filtering algorithms allowing users to sort products by category, cube size (2x2 to 7x7), e-commerce platform (Amazon / Mercado Livre), and price.
* **Responsive Design:** A mobile-first, fluid user interface featuring custom breakpoints, premium card animations, and interactive navigation.
* **Affiliate Architecture:** Clean and compliant affiliate link integration, including automated compliance disclaimers and responsive call-to-action components.
* **E-learning Showcase:** A dedicated "Learn" section designed to promote digital infoproducts and sequential training modules with a conversion-optimized layout.
* **Data-Driven UI:** Components render dynamically from a structured data layer (JSON), making future database/CMS integrations seamless.
